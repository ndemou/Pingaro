Right VBox Spacing Only

This page explains only the layout rows inside the right graph pane's vertical box. The drawing is intentionally not to scale so every row and spacer is easy to read.

The right graph pane has an outer margin of 4px on all sides. Inside that margin, the VBox stacks one 2px spacer, then graph widgets directly one after another. The VBox's own Spacing is zero.

Important Consequences

Top of pane to first spacer = 4px pane margin.
First spacer to first graph widget = 2px VSpacer.
Top of pane to first graph widget = 4px margin + 2px spacer.
Between graph widgets = 0px from the VBox point of view.
The title/header of each graph is not a separate VBox item. It is painted inside the blue CustomWidget. So when we say "space above the graph header", we must decide whether we mean space above the whole CustomWidget, or apparent visual space above the painted header inside it.
