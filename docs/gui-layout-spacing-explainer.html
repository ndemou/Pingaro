6. Why This Is Hard To Tune

Current Mixed Structure

Native layout and custom painting are mixed. The VBox controls only whole graph widgets, while visual subparts are painted manually inside each widget.

VBox row: currentLabel
VBox row: 8px spacer
VBox row: CustomWidget containing header + plot + axes
VBox row: 8px spacer

Cleaner Future Structure

Each graph block could be a real composite. Then header spacing, chart spacing, and axis padding would be controlled by separate layout rows.

GraphBlock
Header row: fixed 18px
Spacer row: fixed 2px
Chart-only CustomWidget
Between blocks: fixed 8px

The key diagnostic question is: are we measuring native space between widgets, or painted empty area inside a widget? In the current design those look similar on screen but come from different code paths.
